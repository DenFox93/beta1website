This generate a very big wordlist.
The personalized password wordlist we generated is about 12387 lines long.
If we know the password policy, we can remove any passwords that do not meet these conditions from our wordlist
Hashcat or John rules, but hydra does not support rules for filtering passwords


sed -ri '/^.{,7}$/d' wordlist.txt                      # need to be longer then 7
sed -ri '/[!-/:-@\[-`\{-~]+/!d' wordlist.txt      # need to contain at least a special character
sed -ri '/[0-9]+/!d' wordlist.txt                       # need to contain at least a number
